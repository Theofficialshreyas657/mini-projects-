# Simon Game

A digital recreation of the classic Simon electronic memory game from the 1980s. Test your memory by following increasingly complex sequences of lights and sounds!

## About the Game

Simon is a memory skill game where players must repeat random sequences of lights and sounds. The game creates a sequence by lighting up one of four colored buttons in a specific order. Players must then repeat the sequence by pressing the buttons in the same order. With each successful round, the sequence gets longer and more challenging.

## How to Play

1. **Start the Game**: Press the start button to begin
2. **Watch**: Simon will light up a sequence of colored buttons with corresponding sounds
3. **Repeat**: Click the buttons in the same order as shown
4. **Progress**: Each round adds one more step to the sequence
5. **Game Over**: One wrong move and it's game over! Try to beat your high score

## Game Rules

- The game starts with a sequence of 1 color
- Each successful round adds 1 more color to the sequence
- You must repeat the entire sequence correctly to advance
- The game speeds up as you progress through levels
- Missing a color or pressing them in the wrong order ends the game

## Features

- 🎨 Four colored buttons (Red, Blue, Green, Yellow)
- 🔊 Unique sound for each color
- 📈 Progressive difficulty with increasing speed
- 🏆 Score tracking
- 🎮 Simple, intuitive interface
- 📱 Responsive design for mobile and desktop

## Controls

- **Mouse/Touch**: Click or tap the colored buttons
- **Start Button**: Begin a new game
- **Restart**: Reset and start over at any time

## Scoring

- Each successful sequence completion adds 1 point to your score
- Try to achieve the highest score possible before making a mistake
- Challenge yourself to beat your previous records!

## Game Difficulty Levels

As you progress, the game becomes more challenging:

- **Rounds 1-4**: Normal speed
- **Rounds 5-8**: Slightly faster
- **Rounds 9-12**: Fast
- **Round 13+**: Very fast - expert level!

## Game Tips & Strategies

- 👀 **Focus**: Eliminate distractions and concentrate fully
- 🎵 **Use Audio**: The sounds help reinforce the pattern
- 🧠 **Chunking**: Break long sequences into smaller groups
- 🔁 **Practice**: The more you play, the better your memory becomes
- ⏱️ **Don't Rush**: Take your time when repeating sequences
